│   │   • Expansion plans require pre-approval (2-week delay)         │   │

│   │   • No direct penalties                                         │   │
│   │                                                                 │   │
│   │   REVIEW STATUS (36-40%)                                        │   │
│   │   ──────────────────────                                        │   │

│   │                                                                 │   │

│   │   • Price increases require justification                       │   │

│   │   • $500/week penalty until compliance                          │   │
│   │   • Social Score -10 (community backlash)                       │   │

│   │   PRICE FIXING                                                  │   │
│   │   ────────────                                                  │   │
│   │   Definition:  Agreeing with competitors to set prices          │   │
│   │   Detection:   30% chance if discussed in communications        │   │
│   │   Penalty:     $1,000 fine + Social Score -12 + investigation   │   │
│   │   Duration:    Scandal marker for 16 weeks                      │   │
│   │                                                                 │   │
│   │   Examples:                                                     │   │
│   │   • "Let's all charge $3.00 per wash"                           │   │
│   │   • "If you don't lower prices, I won't either"                 │   │
│   │   • Coordinated price increases                                 │   │
│   │                                                                 │   │
│   │   ALLOWED:                                                      │   │
│   │   • Independently matching competitor prices                    │   │
│   │   • Public price announcements                                  │   │
│   │   • Responding to market conditions                             │   │
│   │                                                                 │   │

│   │   MARKET ALLOCATION                                             │   │
│   │   ─────────────────                                             │   │
│   │   Definition:  Agreeing to divide customers or territories      │   │
│   │   Detection:   40% chance if discussed in communications        │   │
│   │   Penalty:     $750 fine + Social Score -10                     │   │
│   │   Duration:    Scandal marker for 12 weeks                      │   │
│   │                                                                 │   │
│   │   Examples:                                                     │   │
│   │   • "You take the students, I'll take the families"             │   │
│   │   • "Stay out of Zone A and I'll stay out of Zone B"            │   │
│   │   • "Don't market to my customers"                              │   │
│   │                                                                 │   │
│   │   ALLOWED:                                                      │   │
│   │   • Naturally focusing on certain demographics                  │   │
│   │   • Geographic convenience differences                          │   │
│   │   • Specialization in services                                  │   │
│   │                                                                 │   │

│   │   PREDATORY PRICING                                             │   │
│   │   ────────────────                                              │   │
│   │   Definition:  Pricing below cost to eliminate competitors      │   │
│   │   Detection:   Triggered if prices < variable cost for 4+ weeks │   │
│   │   Penalty:     $500 fine + Social Score -6 + price floor imposed│   │
│   │   Duration:    Scandal marker for 8 weeks                       │   │
│   │                                                                 │   │
│   │   Determination:                                                │   │
│   │   • Regulators compare your prices to variable costs            │   │
│   │   • Intent inferred if competitor exits during period           │   │
│   │   • Market share gains during low-price period scrutinized      │   │
│   │                                                                 │   │
│   │   Defense:                                                      │   │
│   │   • Promotional pricing (limited duration, announced)           │   │
│   │   • Matching competitor's legitimate prices                     │   │
│   │   • Efficiency improvements reducing costs                      │   │
│   │                                                                 │   │

│   │   EXCLUSIVE DEALING (ANTI-COMPETITIVE)                          │   │
│   │   ────────────────────────────────────                          │   │
│   │   Definition:  Forcing vendors to refuse competitors            │   │
│   │   Detection:   50% chance if attempted                          │   │
│   │   Penalty:     $600 fine + Social Score -8 + vendor blacklist   │   │
│   │   Duration:    Scandal marker for 12 weeks                      │   │
│   │                                                                 │   │
│   │   Examples:                                                     │   │
│   │   • Paying vendor to refuse service to competitors              │   │
│   │   • Threatening vendor to drop competitors                      │   │
│   │   • Buying out all vendor capacity to starve competitors        │   │
│   │                                                                 │   │
│   │   ALLOWED:                                                      │   │
│   │   • Normal exclusive contracts (your commitment, not theirs)    │   │
│   │   • Volume discounts without exclusivity demands                │   │
│   │   • Preferred customer relationships                            │   │
│   │                                                                 │   │

│   │   Penalty:     $1,500 fine + Social Score -15 + all parties     │   │

│   │                                                                 │   │
│   │   Examples:                                                     │   │
│   │   • Secret agreement to reduce service quality together         │   │
│   │   • Coordinated refusal to serve certain customers              │   │
│   │   • Sharing confidential customer data for mutual benefit       │   │
│   │   • Joint sabotage of new market entrants                       │   │
│   │                                                                 │   │
│   │   Detection Triggers:                                           │   │
│   │   • Suspiciously similar business decisions                     │   │
│   │   • Whistleblower reports (employees, customers)                │   │
│   │   • Communication intercepts (simulation mechanic)              │   │
│   │   • Statistical analysis of pricing patterns                    │   │

│   │   STAGE 2: PRELIMINARY REVIEW                                   │   │
│   │   ───────────────────────────                                   │   │

│   │   • 40% chance of dismissal if evidence weak                    │   │
│   │                                                                 │   │

│   │   ─────────────────────────────                                 │   │
│   │   Duration: 4 weeks                                             │   │

│   │   • Financial records examined                                  │   │
│   │   • Social Score -5 (major negative publicity)                  │   │
│   │   • Legal fees: $200/week                                       │   │
│   │                                                                 │   │

│   │   ──────────────────────                                        │   │
│   │   Outcomes:                                                     │   │
│   │   • Cleared (30%): Investigation closed, +3 Social Score        │   │
│   │   • Settlement (40%): Reduced penalties, admission of issues    │   │
│   │   • Found Liable (30%): Full penalties applied                  │   │
│   │                                                                 │   │
│   │   STAGE 5: APPEAL (Optional)                                    │   │
│   │   ────────────────────────                                      │   │
│   │   Duration: 4 weeks                                             │   │
│   │   Cost: $500 legal fees                                         │   │
│   │   Success Rate: 25%                                             │   │
│   │   • If successful: Penalties reduced by 50%                     │   │
│   │   • If unsuccessful: +$200 court costs                          │   │
