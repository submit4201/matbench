│                                                                         │
│   ADVANTAGES:                                                           │

│   • Price comparison leverage                                           │
│   • Access to different product lines                                   │
│   • Flexibility during shortages                                        │
│                                                                         │
│   DISADVANTAGES:                                                        │
│   • Cannot reach highest tier with any vendor                           │
│   • No exclusive contract benefits                                      │
│   • Higher administrative overhead                                      │
│   • Vendors may deprioritize your orders                                │

│   RECOMMENDED STRATEGIES:                                               │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                 │   │
│   │   STRATEGY 1: Single Vendor Focus                               │   │

│   │   • Best for: Stable operations, high volume                    │   │
│   │   • Risk: Supply chain vulnerability                            │   │
│   │                                                                 │   │
│   │   STRATEGY 2: Primary + Backup                                  │   │
│   │   • 80% from primary vendor, 20% from backup                    │   │
│   │   • Best for: Risk-conscious operators                          │   │
│   │   • Trade-off: Slower tier progression with primary             │   │
│   │                                                                 │   │
│   │   STRATEGY 3: Specialized Split                                 │   │
│   │   • Different vendors for different product categories          │   │
│   │   • Best for: Quality-focused operations                        │   │
│   │   • Example: GreenClean for detergent, QuickShip for supplies   │   │
│   │                                                                 │   │
│   │   STRATEGY 4: Competitive Bidding                               │   │
│   │   • Regularly switch based on best current pricing              │   │
│   │   • Best for: Price-sensitive, low-margin operations            │   │
│   │   • Risk: Never builds strong relationships                     │   │
